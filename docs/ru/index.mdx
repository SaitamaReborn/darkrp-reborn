## Как это работает

Ваши Lua-скрипты из GMod работают как есть, включая синтаксис GLua и знакомый API:

Ваши аддоны, это просто папки с `.lua`-файлами, помещённые на сервер: **без компиляции, без повторной публикации**.

```lua
-- main.lua, первый аддон в 6 строк
hook.Add("PlayerInitialSpawn", "hello", function(ply)
	timer.Simple(8, function()
		if IsValid(ply) then ply:ChatPrint("Добро пожаловать!") end
	end)
end)
```

Перезагрузка на лету через `addon_reload` в консоли сервера. Сломанный аддон изолируется и логируется, **он никогда не роняет сервер**.

## Начните с простого

Для новичков всё продумано максимально просто: скопируйте `template/addon-lua/` на свой сервер, выполните `addon_reload`, напишите `/hello` в игре. Готово.

## Принципы проекта

- **Ваши аддоны, ваш сервер.** SDK не навязывает правила: создавайте любые профессии и системы.
- **Безопасность прежде всего.** Каждый аддон работает в изолированной песочнице.
- **Совместимость, измеренная на реальных аддонах.** [Подробности о миграции](/ru/compat/migration).
