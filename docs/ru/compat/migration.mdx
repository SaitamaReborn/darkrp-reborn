У вас сервер DarkRP (или MilitaryRP, SCP-RP…) с десятками аддонов? Инструмент `reborn-migrate` переносит **весь сервер сразу**.

## Одна команда

```bash
python tools/reborn-migrate/reborn_migrate.py "C:/путь/к/garrysmod" --out ./migration
```

## Что переносится

| Элемент GMod | Становится | Статус |
|---|---|---|
| `addons/*` (Lua) | аддоны DarkRP Reborn | ✅ автоматически |
| Скриптовые сущности (SENT) | играбельные сущности | ✅ проверено |
| Скриптовое оружие (SWEP) | играбельное оружие | ✅ проверено |
| `darkrpmodification` (профессии, еда) | реплицированный контент | ✅ проверено |
| `sv.db` (профессии, деньги) | `sql_seeds.json` | ✅ экспорт |
| Модели `models/*.mdl` | список + gmod-import | 🟡 полуавтомат |
| Карты `.bsp` | помечаются | 🟡 отдельно |
| Кастомный UI (`surface`/`render`) | переписать на `vgui` | 🔴 вручную |

## Доказательство

Мы перенесли реальный сервер DarkRP, `darkrpmodification` (принтер денег SENT, кастомный AK47 SWEP, профессии) плюс аддон майнинга, и всё **загрузилось без единой ошибки**. Перенесённый принтер денег появляется в игре и работает.

## Проверено на реальном сервере

Полный сервер (71 публичный аддон плюс целый гейммод, 450+ Lua-файлов) прошёл через инструмент: подавляющее большинство аддонов загружается и работает сразу, отчёт точно указывает то немногое, что стоит доработать. Ноль ошибок конвертации.
