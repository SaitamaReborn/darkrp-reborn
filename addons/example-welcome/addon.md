# Welcome message

- **Ident**: `example-welcome`
- **Author**: DarkRP Reborn
- **Version**: 1.0
- **Description**: Greets every joining player (notification + announcement),
  gives 100$ via `/welcome`, reminds players of the command every 5 minutes and
  counts welcomed players (persistent).

Reference addon: it shows the whole contract in one file - events
(`PlayerJoined`, `JobChanged`), a chat command, money, a timer, persistence.
Copy it as your starting point.
